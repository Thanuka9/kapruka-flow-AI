# Kapruka Flow AI Agent Rules

## Python Formatting Rules
- **Rule**: Whenever you modify or add any Python files in the `backend/` directory, you **must** format them using the project's local Ruff formatter before completing your task or pushing changes.
- **Formatter Command**: Run `.venv\Scripts\ruff.exe format app` from the `backend/` directory.
- **Check Command**: Run `.venv\Scripts\ruff.exe format --check app` from the `backend/` directory.
